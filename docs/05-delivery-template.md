# 05. 交付说明模板

下面是一份可以复用的交付说明模板。

## 交付范围

本次交付包含 `N` 个目标的公开评论数据。数据按目标名称归档，每个目标下包含若干视频，每个视频下包含评论列表。

## 文件说明

| 文件 | 用途 |
|---|---|
| `comments_full.json` | 完整评论数据 |
| `by_model/` | 按目标拆分后的 JSON |
| `summary_by_model.csv` | 每个目标的视频数、评论数、状态 |
| `low_volume_notes.csv` | 低量目标说明 |
| `quality_report.md` | 字段、重复、时间范围、异常校验结果 |
| `README.md` | 交付说明 |

## 字段说明

| 字段 | 含义 |
|---|---|
| `model` | 目标归档名 |
| `videoId` | 视频唯一标识 |
| `title` | 视频标题 |
| `videoUrl` | 视频来源 URL |
| `commentId` | 评论唯一标识 |
| `content` | 评论正文 |
| `createTime` | 评论发布时间 |

## 质量校验结果

- 目标总数：`N`
- 有数据目标：`N`
- 0 数据目标：`N`
- 有效评论总数：`N`
- 视频总数：`N`
- 重复评论：`N`
- 缺视频 URL：`N`
- 缺评论时间：`N`
- 时间越界：`N`

## 低量说明

低量目标没有使用无关评论补齐。可能原因包括：

- 公开可见内容较少。
- 目标较冷门。
- 评论时间范围较窄。
- 平台搜索结果有限。
- 名称存在歧义，已按可确认相关内容保留。

## 可回查方式

每条视频保留 `videoUrl`，可通过来源 URL 抽样核对视频内容和评论相关性。

## 注意事项

- 本交付只包含公开可见数据。
- 不包含账号、cookie、浏览器 profile 或任何登录凭证。
- 数据按采集时平台公开可见状态整理，后续平台内容删除、隐藏或排序变化可能导致回查结果变化。
